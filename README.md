##  Tech Stack
- **Framework:** React.js (Vite)
- **Styling:** Tailwind CSS & DaisyUI
- **3D Visuals:** Three.js / React Bits
- **Authentication:** Firebase Auth
- **Routing:** React Router DOM

##  Key Features
- **3D Interactive Hero:** Floating physics-based orbs.
- **Dynamic Adventures:** Data-driven adventure cards from JSON.
- **Secure Auth:** Google & Email/Password login.
- **Private Routes:** Protected adventure details for members.
